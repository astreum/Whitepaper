# Accounts

## Structure

Key: String;
- Address

Value: String;
- Account Details Hash

## Account Details

Key: String;
- Account Details Hash

Value: Stellar Objects;
    1. Counter
    2. Balance


### Balance

The standard unit of value is a Quanta while the smallest unit is a Quark.

Value magnitudes:-

- 10^24: Yottaquark / Quanta
- 10^21: Zettaquark
- 10^18: Exaquark
- 10^15: Petaquark
- 10^12: Teraquark
- 10^9: Gigaquark
- 10^6: Megaquark
- 10^3: Kiloquark
- 10^0: Quark
